## Transforms



###  Transform Syntax#transform-syntax

      div {
     -webkit-transform: scale(1.5);
        -moz-transform: scale(1.5);
         -o-transform: scale(1.5);
            transform: scale(1.5);
     }


**property includes multiple vendor prefixes to gain the best support across all browsers*
